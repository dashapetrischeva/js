# Advanced JavaScript & TypeScript: Logic, Architecture & OOP

This repository showcases 22 practical assignments completed during an intensive 3-month advanced JavaScript and TypeScript training program. The course was strictly focused on production-ready concepts, deep language mechanics, and algorithmic problem-solving.

## 📊 Course Overview
* **Duration:** 3 months
* **Total Lessons:** 24 high-intensity sessions
* **Core Focus:** Clean code, modern ES6+ syntax, asynchronous programming, and Object-Oriented/Functional paradigms.

## 🛠️ Deep Dive Core Concepts Covered

### Language Mechanics & Core Logic
* Advanced data types, strict type conversion, variables, and constants scoping.
* Control flow: Linear execution, complex conditional branching, and optimized loops/cyclic algorithms.
* User interaction basics, data input validation, and algorithmic problem-solving.

### Advanced Data Structures & Language Features
* Deep dive into Functions (scopes, closures, Arrow functions).
* Comprehensive array manipulation using modern built-in methods (`map`, `filter`, `reduce`, etc.).
* Advanced String manipulation and specialized Native Date/Time handling.
* Iterators, generators, and working with modern collection types (`Map`, `Set`).
* Error handling strategies using robustness patterns (`try...catch` blocks).

### Asynchronous JS, Modules & Web APIs
* Advanced asynchronous programming (Promises, `async/await`, API integration, and fetch handling).
* JavaScript Modules (`import`/`export` syntax) for scalable file architecture.
* Working with web scheduling: Timers, `setTimeout`, and `setInterval` contexts.

### Paradigms: Object-Oriented (OOP) & Functional Programming
* Object-Oriented Programming (OOP) fundamentals: Classes, encapsulation, and inheritance mechanisms in JS/TS.
* Functional Programming concepts: Pure functions, immutability, and declarative programming.

### DOM Engine & Interaction
* Mastery of the DOM tree: Advanced manipulation, dynamic element generation, and performant browser events handling.

## 📂 Code Quality & Execution
Every practical task in this repository represents deep hands-on experience in writing maintainable, dry, and clean asynchronous code using industry-standard JavaScript (ES6+) and TypeScript concepts.

    Solution

    Create a class TDate for working with dates in the format "day.month.year". A date is represented as a structure with three fields.     Solution

    Create a class TMoney for working with monetary amounts. The amount should be stored as a dollar equivalent. Implement methods for adding/removing monetary amounts, specifying the amount in hryvnias, and determining the dollar rate at which the amount in hryvnias will increase by 100. Store the dollar rate in a separate field.

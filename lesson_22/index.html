  • Regular Expressions

    Solution

    Task 1. Given an array of strings. Display those that contain digits (use the test method and regular expressions).

    Task 2. Given an array of strings. Display those that do not contain digits.

    Task 3. Given an array of strings. Display those that contain vowel letters.

    Task 4. Given an array of strings. Display those that do not contain vowel letters.

    Task 5. Given an array of strings. Display those that contain either digit 1 or digit 3.

    Task 6. Given a string of text, display all numbers that are in the text.

    Task 7. Given a string of text. Find all punctuation marks that were used.

    Task 8. Given a string of text. Display all components that are separated by punctuation marks.

    Task 9. Given a string of text. Check if it contains a date in the format dd.mm.yyyy (dd- day, mm- month, yyyy- year).

    Task 10. Given a string of text. Count the number of two-digit numbers in the string.

    Task 11. Determine if a string of text can be a bank card number (example: «4142-3433-2323-3434» ). Find all such numbers (while the “-” characters may or may not be present, i.e., «4142343323233434»).

    Task 12. Given a website address. Determine if it is a government website (contains the domain “gov”, but not necessarily at the end). For example: “https://company.gov.ua”

    Task 13. Select all years between 2020 and 2049 from the received message

    Task 14. Given a phone number. Check if this is a Kyivstar operator phone

    Task 15. The user enters a last name and first name in one line separated by a space. Replace the space with a hyphen.

    Task 16. The user enters a date in the format «day.month.year». Get the string representation of the date in the format «month/year»

At startup, we ask the user if they want to read news. If yes, we immediately transition to the Ukr.Net website; if no, we automatically transition to the ukr.net website after 20 seconds. To do this, after 20 seconds of opening, you need to execute the command window.location.href = 'https://www.ukr.net/'
